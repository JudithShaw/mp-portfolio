WEB DESIGN

Producing a successful website depends on many factors which combine to give a good user experience.

The first part of the planning process – creating the hierarchical site map, is important to clearly understand the structure and will enable to user to easily navigate the different pages. The use of a wireframe enables a visual representation of the ideas for the content. GitHub, Atom and GitHub pages are the tools to create the website content and to publish the html and CSS information online.

Choosing a complimentary colour scheme for the website is important. Colour is a powerful tool in graphic design. It can be used to attract attention, organise content, emphasise elements, evoke emotion and help design look aesthetically pleasing. (Dotugo, 2021).

I chose a monochromatic colour scheme for my website. Monochromatic colours are colour schemes that are based on only one colour or tint. One base colour was selected, and all the other colours are variations of the base. The saturation and brightness are chosen in order to make lighter and darker variants. (Torres, 2019)

The colour green was used as the base and the other colours were used to make lighter and darker variants and create a harmonious, visually cohesive look. Green is a cool colour that symbolises the natural world. It is thought to represent freshness, tranquillity and progress and is considered to have a calming effect. (Cherry, 2020). The colour compliments the overall message of my website – The Circle of Life.

The Typography – the visual component of the written work, is important in website design as it helps to maintain a consistency and make it look aesthetically pleasing and professional. Typography helps in making the content attractive and impacts the readability of the website, which all accounts for a positive user experience. (consultancy, 2019).

I chose the sans-serif font KRUB (Demak, 2021). The appearance was clean, easy to read, professional and complimented the theme of my website.

ACCESSIBILITY and INCLUSION - Using inclusive design for a website is important to create accessibility for a greater number of people. In a 2019 web article Web Designer Daniel Schwarz (Schwarz, 2019) states that accessibility is a human right and, in some countries, the law. When it comes to the written word, there are a huge variety of considerations to bear in mind. There is the simplicity of the words (which can affect people with learning difficulties). The typography (which can affect visually impaired users) and the consistency (which can affect people with cognitive disability. Colour contrast is important to enable readability for visually impaired people. To make the website more inclusive I tried to follow these principles and also included alt text to describe the photo.
